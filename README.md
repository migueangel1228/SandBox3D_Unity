# SandBox3D_Unity
Un sandbox 3D simple donde el usuario selecciona una esfera, configura velocidad inicial, ángulo, dirección y masa, la lanza con Rigidbody, observa su trayectoria real con una línea, ve las variables X/Y/Z en tiempo real, consulta la ecuación del movimiento con valores del lanzamiento y puede pausar o ralentizar el tiempo.
